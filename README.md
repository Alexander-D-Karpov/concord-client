# Concord Desktop Client

Cross-platform desktop client for Concord built with Electron and TypeScript.

## Features

- Real-time messaging
- Room management
- Voice chat support
- Cross-platform (Windows, macOS, Linux)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Package

```bash
# macOS
npm run package:mac

# Windows
npm run package:win

# Linux
npm run package:linux
```
